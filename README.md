
### タブ補完を有効化する
```bash
$ terraform -install-autocomplete
```
### 既存のインフラリソースをTerraformリソースに紐づけ
```bash
$ terraform import
```
### Terraformモジュールのテストを実行
```bash
$ terraform test
```
